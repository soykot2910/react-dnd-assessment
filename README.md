
#### Technologies Used
- React
- react-dnd
- react-dnd-html5-backend
- react-modal
- react-split
- shortid
- classNames

#### Project installation
1. Clone Repository
```
$ git clone https://github.com/soykot2910/react-dnd-assessment
$ cd react-dnd-assessment
```
2. Install dependencies

```
$ npm i
```
3. Run Project
```
$ npm start
```
